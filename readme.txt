DeborphanGUI is a tool to find and delete orphaned packages in your system.
It runs on any debian powered distribution.
It allows searching packages with various criteria, then allows to browse 
throug found packages and gives, a brief description about them.
When you run out off place on your hdd, 
it's perfect tool for wipe out some unused libraries and packages.



To run DeborphnGUI you should install some packages first:

python2.4
python-wxgtk2.6
libwxgtk2.6-0
libwxbase2.6-0
apt
deborphan
xterm

...but should work with other versions too.
